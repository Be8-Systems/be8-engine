# be8-engine
Be8 uses a Elliptic Curve Diffie-Hellman 384 bit prime curve encryption. 

## usage
The constructer takes one parameter the accID. 

```javascript
const be8 = new Be8(1);
```

## hasKeys()

Checks if the object already generated keys and if they are stored.
Returns a boolean.

```javascript
be8.hasKeys();
```
